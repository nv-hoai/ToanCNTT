THUAT TOAN: Phan tich thua so nguyen to (Trial Division)
DO PHUC TAP: O(sqrt(n))

PSEUDO CODE:
function primeFactorization(n):
    factors = []
    exponents = []
    
    // Xu ly uoc so 2
    count = 0
    while n % 2 == 0:
        count = count + 1
        n = n / 2
    if count > 0:
        factors.append(2)
        exponents.append(count)
    
    // Xu ly cac uoc so le
    i = 3
    while i * i <= n:
        count = 0
        while n % i == 0:
            count = count + 1
            n = n / i
        if count > 0:
            factors.append(i)
            exponents.append(count)
        i = i + 2
    
    // Neu n > 1 thi n la so nguyen to
    if n > 1:
        factors.append(n)
        exponents.append(1)
    
    return (factors, exponents)

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Phân tích thừa số nguyên tố (kèm số mũ)

Input:
• Số nguyên dương n

Khởi tạo:
• factors = []
• exponents = []

Lặp:
1. Xử lý ước số 2:
   - count = 0
   - Trong khi n chia hết cho 2:
     * count++
     * n = n / 2
   - Nếu count > 0:
     * Thêm 2 vào factors
     * Thêm count vào exponents

2. Xử lý các ước lẻ:
   - Với i từ 3 đến √n (bước nhảy 2):
     * count = 0
     * Trong khi n chia hết cho i:
       - count++
       - n = n / i
     * Nếu count > 0:
       - Thêm i vào factors
       - Thêm count vào exponents

3. Nếu n > 1:
   - Thêm n vào factors
   - Thêm 1 vào exponents

Ví dụ: n = 360 = 2³ × 3² × 5
• factors = [2, 3, 5]
• exponents = [3, 2, 1]
