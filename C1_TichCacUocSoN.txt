THUAT TOAN: Tim tich cac uoc so cua n
DO PHUC TAP: O(sqrt(n))

PSEUDO CODE:
function productOfDivisors(n):
    count = countDivisors(n)
    product = n^(count / 2)
    return product

function countDivisors(n):
    count = 0
    for i = 1 to sqrt(n):
        if n % i == 0:
            if i * i == n:
                count = count + 1
            else:
                count = count + 2
    return count

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Tính tích các ước số

Input:
• Số nguyên dương n

Công thức:
• Tích = n^(k/2), với k là số lượng ước số

Bước 1: Đếm số ước
• k = countDivisors(n)

Bước 2: Tính tích
• product = n^(k/2)

Giải thích:
- Ước số xuất hiện theo cặp (i, n/i)
- Tích mỗi cặp = i × (n/i) = n
- Nếu có k ước → có k/2 cặp → tích = n^(k/2)

Ví dụ: n = 6
• Ước: {1, 2, 3, 6} → k = 4
• Tích = 1×2×3×6 = 36 = 6^(4/2) = 6²
