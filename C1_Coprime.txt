THUAT TOAN: Euler's Totient Function (Ham Phi Euler)
DO PHUC TAP: O(sqrt(n))

PSEUDO CODE:
function eulerPhi(n):
    result = n
    p = 2
    while p * p <= n:
        if n % p == 0:
            while n % p == 0:
                n = n / p
            result = result - result / p
        p = p + 1
    if n > 1:
        result = result - result / n
    return result


THUAT TOAN: Greatest Common Divisor (Thuat toan Euclidean)
DO PHUC TAP: O(log(min(a, b)))

PSEUDO CODE:
function gcd(a, b):
    while b != 0:
        temp = b
        b = a % b
        a = temp
    return a


THUAT TOAN: Tim cac so nguyen to cung nhau voi n
DO PHUC TAP: O(n * log(n))

PSEUDO CODE:
function findCoprime(n):
    coprimes = []
    for i = 1 to n-1:
        if gcd(i, n) == 1:
            coprimes.append(i)
    return coprimes

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Hàm Phi Euler φ(n)

Input:
• Số nguyên dương n

Khởi tạo:
• result = n
• p = 2

Lặp:
1. Trong khi p² ≤ n:
   - Nếu n chia hết cho p:
     * Chia n cho p đến khi không chia hết
     * result = result - result/p
   - p = p + 1

2. Nếu n > 1:
   - result = result - result/n

3. Trả về result

Công thức: φ(n) = n × ∏(1 - 1/p) với p là ước nguyên tố của n

Ý nghĩa: φ(n) = số lượng số nguyên tố cùng nhau với n trong [1, n-1]
