THUAT TOAN: Uoc chung lon nhat (Euclidean Algorithm - GCD)
DO PHUC TAP: O(log(min(a, b)))

PSEUDO CODE:
function gcd(a, b):
    while b != 0:
        temp = b
        b = a % b
        a = temp
    return a


THUAT TOAN: Boi chung nho nhat (LCM)
DO PHUC TAP: O(log(min(a, b)))

PSEUDO CODE:
function lcm(a, b):
    return (a * b) / gcd(a, b)

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Euclid (Ước chung lớn nhất)

Input:
• Hai số nguyên dương a, b

Lặp:
1. Trong khi b ≠ 0:
   - temp = b
   - b = a mod b
   - a = temp

2. Trả về a

Ý tưởng: GCD(a, b) = GCD(b, a mod b)

Ví dụ GCD(48, 18):
- 48 mod 18 = 12 → GCD(18, 12)
- 18 mod 12 = 6 → GCD(12, 6)
- 12 mod 6 = 0 → Kết quả: 6

---

Thuật toán: Bội chung nhỏ nhất

Input:
• Hai số nguyên dương a, b

Công thức:
• LCM(a, b) = (a × b) / GCD(a, b)

Giải thích:
- Tích a × b chứa tất cả thừa số
- Chia GCD để loại bỏ phần trùng lặp
