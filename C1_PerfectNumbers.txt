THUAT TOAN: Tim so hoan hao (Perfect Number Search)
DO PHUC TAP: O(n * sqrt(n))

PSEUDO CODE:
function isPerfect(n):
    if n <= 1:
        return false
    
    sum = 1
    for i = 2 to sqrt(n):
        if n % i == 0:
            sum = sum + i
            if i != n / i:
                sum = sum + n / i
    
    return (sum == n)

function findPerfectNumbers(limit):
    perfects = []
    
    for num = 2 to limit:
        if isPerfect(num):
            perfects.append(num)
    
    return perfects

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Tìm số hoàn hảo

Input:
• Số nguyên dương n

Định nghĩa: Số hoàn hảo là số bằng tổng các ước thực sự của nó

Kiểm tra:
1. Nếu n ≤ 1: trả về false

2. Khởi tạo sum = 1

3. Với i từ 2 đến √n:
   - Nếu n chia hết cho i:
     * sum = sum + i
     * Nếu i ≠ n/i: sum = sum + n/i

4. Trả về (sum == n)

Ví dụ:
• n = 6: ước thực sự là {1, 2, 3}, tổng = 6 → số hoàn hảo
• n = 28: ước thực sự là {1, 2, 4, 7, 14}, tổng = 28 → số hoàn hảo

Lưu ý: Số hoàn hảo rất hiếm (6, 28, 496, 8128, ...)
