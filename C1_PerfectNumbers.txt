THUAT TOAN: Tim so hoan hao (Perfect Number Search)
DO PHUC TAP: O(n * sqrt(n))

PSEUDO CODE:
function isPerfect(n):
    if n <= 1:
        return false
    
    sum = 1
    for i = 2 to sqrt(n):
        if n % i == 0:
            sum = sum + i
            if i != n / i:
                sum = sum + n / i
    
    return (sum == n)

function findPerfectNumbers(limit):
    perfects = []
    
    for num = 2 to limit:
        if isPerfect(num):
            perfects.append(num)
    
    return perfects

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Tìm tất cả số hoàn hảo đến giới hạn cho trước:
- Chuẩn bị danh sách rỗng để lưu các số hoàn hảo
- Duyệt qua từng số từ 2 đến giới hạn (limit)
- Với mỗi số, kiểm tra xem có phải số hoàn hảo không bằng cách:
  * Tính tổng tất cả ước số thực sự (không tính chính nó)
  * So sánh tổng với chính số đó
  * Nếu bằng nhau → là số hoàn hảo, thêm vào danh sách
- Trả về danh sách tất cả số hoàn hảo tìm được

Lưu ý về số hoàn hảo:
- Các số hoàn hảo rất hiếm: 6, 28, 496, 8128, ...
- Đến nay mới biết 51 số hoàn hảo
- Tất cả số hoàn hảo đã biết đều là số chẵn
- Chưa biết có tồn tại số hoàn hảo lẻ hay không
