THUAT TOAN: Phan tich so thanh thua so nguyen to (Prime Factorization)
DO PHUC TAP: O(sqrt(n))

PSEUDO CODE:
function primeFactorization(n):
    factors = []
    
    // Kiem tra uoc so 2
    while n % 2 == 0:
        factors.append(2)
        n = n / 2
    
    // Kiem tra cac uoc so le tu 3 tro di
    i = 3
    while i * i <= n:
        while n % i == 0:
            factors.append(i)
            n = n / i
        i = i + 2
    
    // Neu n > 1 thi n la so nguyen to
    if n > 1:
        factors.append(n)
    
    return factors


THUAT TOAN: Tim so nguyen to gan nhat
DO PHUC TAP: O(n * sqrt(n))

PSEUDO CODE:
function isPrime(n):
    if n < 2:
        return false
    if n == 2:
        return true
    if n % 2 == 0:
        return false
    for i = 3 to sqrt(n) step 2:
        if n % i == 0:
            return false
    return true

function findNearestPrime(target):
    if target <= 2:
        return 2
    
    lower = target - 1
    upper = target + 1
    
    while lower >= 2 or upper < infinity:
        if lower >= 2 and isPrime(lower):
            return lower
        if isPrime(upper):
            return upper
        lower = lower - 1
        upper = upper + 1
    
    return 2

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Phân tích thừa số nguyên tố

Input:
• Số nguyên dương n

Khởi tạo:
• factors = []

Lặp:
1. Trong khi n chia hết cho 2:
   - Thêm 2 vào factors
   - n = n / 2

2. Với i từ 3 đến √n (bước nhảy 2):
   - Trong khi n chia hết cho i:
     * Thêm i vào factors
     * n = n / i

3. Nếu n > 1:
   - Thêm n vào factors (n là thừa số nguyên tố cuối)

4. Trả về factors

Ví dụ: n = 60
• 60 = 2² × 3 × 5
• factors = [2, 2, 3, 5]
