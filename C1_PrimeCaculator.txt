THUAT TOAN: Kiem tra so nguyen to (Primality Test)
DO PHUC TAP: O(sqrt(n))

PSEUDO CODE:
function isPrime(n):
    if n < 2:
        return false
    if n == 2:
        return true
    if n % 2 == 0:
        return false
    
    for i = 3 to sqrt(n) step 2:
        if n % i == 0:
            return false
    
    return true


THUAT TOAN: Tim so nguyen to trong doan
DO PHUC TAP: O((b - a) * sqrt(b))

PSEUDO CODE:
function findPrimesInRange(a, b):
    primes = []
    
    for num = a to b:
        if isPrime(num):
            primes.append(num)
    
    return primes

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Kiểm tra số nguyên tố

Input:
• Số nguyên n

Kiểm tra:
1. Nếu n < 2: trả về false
2. Nếu n = 2: trả về true
3. Nếu n chẵn: trả về false

Lặp:
• Với i từ 3 đến √n (bước nhảy 2):
  - Nếu n chia hết cho i: trả về false

• Trả về true

Giải thích:
- Chỉ kiểm tra số lẻ vì số chẵn đã loại
- Chỉ kiểm tra đến √n vì nếu n có ước > √n thì cũng có ước < √n
