THUAT TOAN: Dem so luong uoc so cua n
DO PHUC TAP: O(sqrt(n))

PSEUDO CODE:
function countDivisors(n):
    count = 0
    
    for i = 1 to sqrt(n):
        if n % i == 0:
            if i * i == n:
                count = count + 1
            else:
                count = count + 2
    
    return count

MÔ TẢ BẰNG NGÔN NGỮ TỰ NHIÊN:

Thuật toán: Đếm số lượng ước số

Input:
• Số nguyên dương n

Khởi tạo:
• count = 0

Lặp:
• Với i từ 1 đến √n:
  - Nếu n chia hết cho i:
    * Nếu i² = n: count = count + 1
    * Ngược lại: count = count + 2

• Trả về count

Giải thích:
- Ước số xuất hiện theo cặp (i, n/i)
- Nếu i² = n thì i và n/i trùng nhau → chỉ đếm 1

Ví dụ: n = 12
• Ước: {1,12}, {2,6}, {3,4} → tổng cộng 6 ước
